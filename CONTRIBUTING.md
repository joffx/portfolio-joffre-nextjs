# Guía de Contribución

Gracias por tu interés en contribuir a este proyecto. Sin embargo, este es un **repositorio privado y propietario** de FasteryDev, LLC.

## Política de Contribuciones

Este proyecto es mantenido internamente por FasteryDev, LLC. Las contribuciones externas no están permitidas sin autorización previa y explícita.

## Si Tienes una Solicitud

Si tienes una sugerencia, reporte de bug o solicitud de funcionalidad:

1. **Issues**: Puedes crear un issue en este repositorio para:
   - Reportar bugs
   - Sugerir nuevas funcionalidades
   - Hacer preguntas sobre el proyecto

2. **Contacto Directo**: Para consultas más formales o propuestas de colaboración:
   - Email: joffre@fastery.dev
   - Website: https://fastery.dev

## Estándares de Código

Si tienes autorización para contribuir, por favor sigue estos estándares:

### Estilo de Código
- Usa TypeScript para todo el código nuevo
- Sigue las reglas de ESLint configuradas
- Usa Prettier para formateo (si está configurado)
- Escribe código limpio y legible

### Commits
- Usa mensajes de commit descriptivos
- Sigue el formato: `tipo(scope): descripción`
- Tipos comunes: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`

### Pull Requests
- Crea una rama descriptiva para tu cambio
- Asegúrate de que todos los tests pasen
- Actualiza la documentación si es necesario
- Describe claramente tus cambios en el PR

## Estructura del Proyecto

```
portfolio-joffre-nextjs/
├── app/              # App Router de Next.js
├── components/        # Componentes React
├── lib/              # Utilidades
├── public/           # Archivos estáticos
└── types/            # Tipos TypeScript
```

## Desarrollo Local

1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Ejecuta en desarrollo: `npm run dev`
4. Ejecuta el linter: `npm run lint`
5. Construye para producción: `npm run build`

## Reportar Problemas

Si encuentras un bug o problema:

1. Verifica que no haya sido reportado ya
2. Crea un nuevo issue con:
   - Descripción clara del problema
   - Pasos para reproducir
   - Comportamiento esperado vs actual
   - Información del entorno

## Preguntas

Si tienes preguntas sobre el proyecto, puedes:
- Crear un issue con la etiqueta `question`
- Contactar directamente: joffre@fastery.dev
